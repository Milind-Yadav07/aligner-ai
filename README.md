## 📋 Overview

Aligner AI is a cutting-edge web application that leverages **Large Language Models** and **AI** to intelligently process, analyze, and align documents. It supports multiple document formats including PDF, Word documents, and more, making it easy to work with various file types.

**🌐 Live Demo:** [https://aligner-ai.vercel.app](https://aligner-ai.vercel.app)

## ✨ Key Features

- 📄 **Multi-format Document Support** - Process PDF, Word, and other document types
- 🤖 **AI-Powered Analysis** - Uses Google Generative AI for intelligent document processing
- 🔗 **LangChain Integration** - Advanced language model orchestration and chaining
- 💾 **Client-Side Storage** - LocalStorage support via Dexie for offline functionality
- 📊 **PDF Generation** - Export results as formatted PDF documents
- 🎨 **Modern UI** - Beautiful, responsive interface with Tailwind CSS
- ⚡ **Fast Performance** - Optimized Next.js 16 with React 19

## 🛠️ Tech Stack

### Frontend
- **⚛️ React** 19.2.4 - UI library
- **🎯 Next.js** 16.2.4 - React framework with API routes
- **🎨 TailwindCSS** 4.0 - Utility-first CSS framework
- **📦 Lucide React** - Icon library
- **✅ Zod** - TypeScript-first schema validation

### AI & Language Models
- **🔮 Google Generative AI** - Gemini API integration
- **🔗 LangChain** - LLM framework and orchestration
- **🧠 LangChain Google GenAI** - Google integration for LangChain

### Document Processing
- **📑 PDF.js** - PDF viewer and parser
- **📄 jsPDF** - PDF generation
- **📊 jsPDF AutoTable** - PDF table formatting
- **📘 Mammoth** - Word document conversion

### Utilities & Dev Tools
- **📝 ESLint** - Code linting
- **🔤 clsx** - Conditional class names
- **🎯 Tailwind Merge** - Tailwind class utilities
- **🗄️ Dexie** - IndexedDB wrapper for client-side storage

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ or latest
- npm, yarn, pnpm, or bun

### 📦 Installation & Setup

1. Clone the repository
```bash
# Clone the repository
git clone https://github.com/Milind-Yadav07/aligner-ai.git

# Install dependencies
npm install

# Start the development server
npm run dev
```
